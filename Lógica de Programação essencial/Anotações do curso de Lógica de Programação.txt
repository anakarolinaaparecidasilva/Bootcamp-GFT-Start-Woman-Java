LÓGICA DE PROGRAMAÇÃO ESSENCIAL
É utilizar a lógica na programação de computadores.

- metacognição: pensar como você pensa.

Exemplo:
Precisa comprar 2 camisetas, cada uma custa R$50,00. Porém comprando 2, ganha 20% de desconto. Qual valor do desconto e quanto irá pagar pelas duas camisetas?
Resolução: TOTAL: 2xR$50,00=R$100,00
VALOR DO DESCONTO: 20%=20/100=0,2=R$20,00
IRÁ PAGAR: R$100,00-R$20,00=R$80,00

*LÓGICA:
2xR$50,00=R$100,00
20% de desconto, irá pagar 80% de R$100,00=R$80,00

FLUXOGRAMA: 
usado para representar graficamente o algoritmo.
Diagrama de blocos: representa o método do fluxograma.

VARIÁVEIS: 
objeto para reter ou representar um valor ou expressão, pode ser alterado durante a execução do algoritmo.

CONSTANTES: 
valor imutável, não pode ser alterado.

Link Flowgorithm:
http://www.flowgorithm.org -> linguagem em português.

EXPRESSÕES ARITMÉTICAS: 
utiliza operadores e funções aritméticas.
- Operadores aritméticos: soma (+), subtração (-), multiplicação (*), divisão (/), (^), (%).

EXPRESSÕES LITERAIS: 
usada na atribuição de valor.
Ao atribuir = 
e para valor usar ""

OPERADORES RELACIONAIS: 
retornam valores lógicos.
São eles: maior (>), maior ou igual (>=), menor (<), menor ou igual (<=), igual (==), diferente (!=).

CONCATENAÇÃO:
designa operação de nir o conteúdo de duas strings (sequência de caracteres). Ex: nome e sobrenome.

ESTRUTURA DE REPETIÇÃO:
permite executar mais de uma vez o mesmo comando, de acordo com uma condição ou contador.

LINGUAGEM DE PROGRAMAÇÃO:
é uma linguagem para gerar programas.
Serve como meio de comunicação entre máquinas e humanos. 
Existem dois tipos:
- Alto nível: mais se aproxima da linguagem humana.
- Baixo nível: mais se aproxima da máquina.

Há as linguagens:
- Compiladas: digita o código e o compilador vai transformar em executável, ou seja, traduz para o código de máquina. Exemplos: DELPHI, C++.

- Interpretadas: código executado por um programa. Exemplos: JavaScript, PHP, Phyton.

SOFTWARE PORTUGOL: usa uma pseduolinguagem para desenvolver algoritmos, resolvendo o problema em si.
