DOMINANDO IDES JAVA

JAVA
é uma linguagem de programação orientada o objetos e é intrepretada. É compilada para um bytecode que é interpretado por uma máquina virtual (JVM).

JAVA SE: base da plataforma

JDK (Kit de desenvolvimento JAVA): compila .java para .class
OpenJDK: gratuito, open source.
JDK Oracle: paga licença.

JRE: roda programa no PC. Máquina virtual executa o bytecode .class
JVM: executa. Possui bibliotecas padrões do JAVA.

IDE (Ambiente de Desenvolvimento Integrado)
É um software que combina ferramentas comuns em única interface gráfica de usuário (GUI), facilitando o desenolvimento de aplicações.

JDK faz parte do funcionamento das IDE's.

Há algumas IDE's como:
ECLIPSE: open source
IntelliJ IDEA: tem versão paga e gratuita
